**REXORIK** — это мощный бот для обработки заявок, автоматизации задач и управления процессами.


---

## ⚡ Возможности

### 📩 Работа с заявками

* Приём пользовательских запросов
* Система тикетов
* Автоответы и уведомления

### ⚙️ Автоматизация

* Обработка действий без участия человека
* Интеграция с API
* Гибкие сценарии работы

### 📊 Контроль и аналитика

* Сбор статистики
* Логирование действий
* Отслеживание активности

### 🔐 Безопасность

* Whitelist доступ
* Ограничение по ролям / ID
* Защита от спама
